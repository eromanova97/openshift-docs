== Asynchronous errata updates

Security, fixed issues, and enhancement updates for {product-title} {product-version} are released as asynchronous errata through the Red{nbsp}Hat Network. This includes updates for the in-cluster monitoring stack and other components.

For more information about monitoring updates, see link:https://docs.redhat.com/en/documentation/openshift_container_platform/4.19/html-single/release_notes/index#asynchronous-errata-updates[Asynchronous errata updates].
