= Accessing the Alerting UI

[role="_abstract"]
You can access the Alerting UI througweb console.

.Procedure

* In the web console, go to *Observe* -> *Alerting*. The three main pages in the Alerting UI in this perspective are the *Alerts*, *Silences*, and *Alerting rules* pages.
