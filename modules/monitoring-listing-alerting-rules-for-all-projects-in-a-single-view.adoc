= Listing alerting rules for all projects in a single view

[role="_abstract"]
As a cluster administrator, you can list alerting rules for core platform and user-defined projects together in a single view.

.Prerequisites

* You have access to the cluster as a user with the following role:
** The `cluster-admin` cluster role in {ocp} and {product-rosa-hcp}.
** The `dedicated-admin` role in {product-rosa} and {product-dedicated}.
* You have installed the {oc-first}.

.Procedure

. In the web console, go to *Observe* -> *Alerting* -> *Alerting rules*.

. Select the *Platform* and *User* sources in the *Filter* drop-down menu.
+
[NOTE]
====
The *Platform* source is selected by default.
====
